# 2D-game-C-SDL
This game is created with C and SDL in 2012 when i was in second year of university, i found it and wanna share it with you, enjoy

## Requirement

- [SDL]([https://nodejs.org/en/](https://wiki.libsdl.org/Installation))

## Setup

- clone the project

```shell script
$> git clone https://github.com/Thegitway/2D-game-C-SDL.git
```
